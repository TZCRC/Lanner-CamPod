# SDE-CamPod
Build information and development for the Cessna camera pod

## Equipment

Each 'pod' consists of:
* DSLR: Nikon D5600 + fixed lens;
* Power system:
    * Tethertools battery;
    * USB battery bank;
* Controller / logger: Raspberry Pi 3 (or Zero), connected to:
    * IMU: BerryGPS-IMU
    * Optocoupler
* Cables:
    * USB Pi -> camera
    * Shutter release from optocoupler -> camera
